# 📝 Noted — Modern Note Taking App

A clean, minimal, dark-themed note taking app built with vanilla HTML, CSS, and JavaScript.

## Features
- 🌙 Dark mode interface
- ✏️ Create, edit, and delete notes
- 🔍 Real-time search
- 💾 Auto-save to localStorage
- ⚡ Zero dependencies — pure vanilla JS

## Getting Started
Just open `index.html` in your browser — no build step needed!

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- localStorage for persistence
